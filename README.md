# AWS Minecraft Server Host Website
### This is a web application that allows users to start, stop, and monitor Minecraft servers hosted on AWS EC2 through a simple interface.
<img width="1488" height="837" alt="image" src="https://github.com/user-attachments/assets/55e2d5fc-b9f9-45d4-a761-06e2f158787a"/>

---

## 👾Tech Stack:

Front-end: React, Next.js 

Cloud & back-end: AWS API Gateway, Lambda, VPC, EC2, IAM

Infrastructure and scripting: Linux, Bash 

UI libraries: ShadCN, Reactbits.DEV
